# PrototUnity

This is a Unity package with a bunch of useful assets (scripts, textures, prefabs, etc) for fast and easy game prototyping.

## How to use

1. Open Unity project
2. Go to `Pacakge Manager -> Install package from git URL...` and type `https://github.com/ivandev0/PrototUnity.git?path=Packages/com.ivan_dev0.prototunity`
